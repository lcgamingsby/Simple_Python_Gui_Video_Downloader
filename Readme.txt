Before running the main.py file, you must download and install the "FFmpeg" package.

To do this, you can follow these steps:
Note : For Windows user only(i only have windows device).

1. Download ffmpeg from the official FFmpeg website.
2. Extract the downloaded ZIP file.
3. Add the bin folder from the extracted content to your system's PATH:
    a. Open "Environment Variables" on Windows.
    b. In "System Variables," find the Path variable and edit it.
    c. Add the path to the bin folder from ffmpeg.
    for example "C:\Program Files\ffmpeg\bin"
4. check the version using "ffmpeg -version"
5. if done, change the ffmpeg path on main.py.

to use the program just run the main.py
